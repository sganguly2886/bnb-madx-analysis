# Numerical Results (CSV Files)

This folder contains all numerical outputs generated by the Python analysis scripts. These CSV files store fitted beam parameters, optical functions, and predicted beam sizes in machine-readable format.

They serve as:
- The quantitative backbone of all figures
- Inputs for cross-comparisons and validation
- Archival records of every quadrupole scan study

---

## Categories of CSV Files

### **1. Beta Function Results**
Examples:
- `betax_vs_Q873_current.csv`
- `betax_vs_HQ874_current_posPol.csv`
- `betax_vs_HQ875_current_flipPol.csv`

**Contents:**
- βx values at MW873, MW875, MW876
- As a function of quadrupole current

**Used to:**
- Identify focusing trends
- Locate optics minima and extrema

---

### **2. Dispersion Results**
Examples:
- `dispersion_vs_Q873_current.csv`
- `dispersion_vs_HQ874_current_posPol.csv`

**Contents:**
- Dx and Dy at the wire chambers
- Quadrupole-dependent dispersion response

**Used to:**
- Evaluate chromatic contributions to beam size
- Interpret σx behavior in conjunction with βx

---

### **3. Emittance Results**
Examples:
- `emittance_vs_Q873_current.csv`
- `emittance_vs_Q874_current_posPol_pi-mm-mrad.csv`
- `emittance95_geometric_Q875_flipPol_pi-mm-mrad.csv`

**Contents:**
- Geometric emittance (RMS and 95%)
- Normalized emittance where applicable

**Used to:**
- Track transverse beam quality
- Check consistency of fitted ε across scans

---

### **4. Momentum Spread Results**
Examples:
- `momentum_spread_vs_Q873_current.csv`
- `momentum_spread_vs_Q874_current_with_Dp.csv`

**Contents:**
- RMS fractional momentum spread σδ

**Used to:**
- Quantify dispersion-driven beam growth
- Validate chromatic behavior inferred from MAD-X

---

### **5. Predicted Beam Sizes and Target Spot**
Examples:
- `madx_pred_sigma_vs_current.csv`
- `target_sigma_prediction_scaled_D_Q873.csv`
- `target_sigma_prediction_scaled_D_Q874.csv`
- `target_sigma_prediction_scaled_D_Q875.csv`

**Contents:**
- Predicted σx and σy at MWs and at the BNB target
- Effective 2D beam spot area

**Used to:**
- Rank optimal quadrupole settings
- Identify global vs local minima in beam spot size

---
